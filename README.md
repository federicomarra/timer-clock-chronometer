# timer-clock-chronometer
### Programmation Laboratory  C++ Unifi

##### C++ Clock program with Timer and Chronometer features, all with multiple visualisation formats.


| INSTRUCTION | TIMER | CLOCK | CHRONOMETER |
|-------------|:-----:|:-----:|:-----------:|
| start       |   S   |       |      V      |
| stop        |   T   |       |    SPACE    |
| reset       |   R   |       |      B      |
| change view |   W   |   K   |      N      |
| +1s         |  UP   |       |             |
| -1s         | DOWN  |       |             |
| exit        |  ESC  |  ESC  |     ESC     |
